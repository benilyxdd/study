std::all_of

Test condition on all elements in range

Returns true if pred returns true for all the elements in the range [first,last) or if the range is empty, and false otherwise.

std::all_of(foo.begin(), foo.end(), [](int i) {return i % 2;})


std::any_of

Test if any element in range fulfills condition

Returns true if pred returns true for any of the elements in the range [first,last), and false otherwise. If [first,last) is an empty range, the function returns false.

std::any_of(foo.begin(), foo.end(), [](int i) {return i == 0;})


std::none_of

Test if no elements fulfill condition

Returns true if pred returns false for all the elements in the range [first,last) or if the range is empty, and false otherwise.

std::none_of(foo.begin(), foo.end(), [](int i) {return i == 0;})


std::for_each

Apply function to range

Applies function fn to each of the elements in the range [first,last).

std::find

Find value in range

Returns an iterator to the first element in the range [first,last) that compares equal to val. If no such element is found, the function returns last. The function uses operator== to compare the individual elements to val.

vector‹int›::iterator it = find (myvector.begin(), myvector.end(), 30);
